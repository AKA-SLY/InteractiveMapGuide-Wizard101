Contribute Wizard101 game data

Drop JSON exports for spells, items, or other categories into the /Json folder at the project root. The app automatically ingests every .json array in that folderâ€”no code edits required.

  1. Place your JSON files inside /Json. Each file may contain one array or multiple objects representing spells, gear, treasure cards, etc.
  2. Keep fields descriptive (for example: name, school, pip_cost/pips, accuracy, effect, bonuses_raw, slot, categories, wiki_url).
  3. Run or refresh the app. The Vite build picks up new JSON files automatically through the import.meta.glob loader.

For large dumps, consider splitting files by category (for example gear_items.json and school_spells.json) to keep them organized.
